(de)Motivator.js

API Documentation

This is the API documentation for the (de)Motivator.js library.
Ever hated your user base so much that you just want to throw every insult imagineable to them? Us too. That's why we had the idea to extend the already existing power of (de)Motivator, and package it into a JavaScript library. You can download the library right now using npm and use it in your own projects.

Getting Started
To get started, install the package via npm. Youll need node.js for this operation. You can find the documentation for the package here. If you're using a module bundler, such as webpack, rollup, parcel, or any other module bundler, you can directly link to the package via a script tag. If you are not, you must reference the package via a "require" statement in your main JavaScript file.

All Done
Now, you can use the "generateInsult()" function and "insult" id to generate as many insults as you want.
